## 1. Naming Convention

- snake_case
    - folders
    - files
- camelCase
  - functions
  - variables
  - constants
- PascalCase
  - classes
  - extensions
  - mixins
> Enum
>   - names : PascalCase
>   - parameters : camelCase

## 2. Write Every Type Everywhere

`add(a,b) => a+b;` -> wrong  
`int add(int a, int b) => a+b;` -> right
